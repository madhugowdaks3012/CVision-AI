## ğŸ§  How It Works

1. Extracts text from uploaded PDF resume  
2. Splits resume into meaningful text chunks  
3. Converts resume and job description into vector embeddings  
4. Computes cosine similarity  
5. Returns the average of top matching segments  
6. Displays candidate fit score with feedback  

This improves accuracy compared to simple TF-IDF keyword matching.
